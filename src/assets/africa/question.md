🌍 รายการคำถาม-คำตอบ: เกมทายเมืองหลวงโซนแอฟริกา
ข้อที่ 1: ประเทศอียิปต์ (Egypt)

คำถาม: เมืองหลวงของประเทศอียิปต์ที่มีมหาพีระมิดกีซาตั้งอยู่คือเมืองอะไร?

คำใบ้: เป็นเมืองที่ใหญ่ที่สุดในโลกอาหรับ

คำตอบ: ไคโร (Cairo)
//
ข้อที่ 2: ประเทศแอฟริกาใต้ (South Africa)

คำถาม: เมืองหลวงที่เป็นศูนย์กลางการบริหารของประเทศแอฟริกาใต้คือเมืองอะไร?

คำใบ้: เป็นที่ตั้งของอาคาร Union Buildings และรูปปั้นเนลสัน แมนเดลา

คำตอบ: พริทอเรีย (Pretoria)
//
ข้อที่ 3: ประเทศเคนยา (Kenya)

คำถาม: เมืองหลวงที่มีฉายาว่า "นครหลวงสีเขียวในแสงแดด" และมีอุทยานสัตว์ป่าติดเขตเมืองคือเมืองอะไร?

คำใบ้: ชื่อเมืองมีความหมายว่า "แหล่งน้ำเย็น" ในภาษามาไซ

คำตอบ: ไนโรบี (Nairobi)
//
ข้อที่ 4: ประเทศโมร็อกโก (Morocco)

คำถาม: เมืองหลวงของโมร็อกโกที่เป็นเมืองมรดกโลกและมีหอคอยฮัสซันตั้งอยู่คือเมืองอะไร?

คำใบ้: ตั้งอยู่ริมชายฝั่งมหาสมุทรแอตแลนติก

คำตอบ: ราบัต (Rabat)
//
ข้อที่ 5: ประเทศเอธิโอเปีย (Ethiopia)

คำถาม: เมืองที่เป็นที่ตั้งของสำนักงานใหญ่สหภาพแอฟริกา (African Union) คือเมืองอะไร?

คำใบ้: เป็นเมืองหลวงที่ตั้งอยู่บนพื้นที่สูงเป็นอันดับต้นๆ ของทวีป

คำตอบ: แอดดิสอาบาบา (Addis Ababa)
//
ข้อที่ 6: ประเทศไนจีเรีย (Nigeria)

คำถาม: เมืองหลวงที่ถูกสร้างขึ้นใหม่ใจกลางประเทศไนจีเรียเพื่อความสมดุลทางการเมืองคือเมืองอะไร?

คำใบ้: มีแลนด์มาร์คสำคัญคือหินแกรนิตขนาดยักษ์ "หินซูม่า"

คำตอบ: อาบูจา (Abuja)
//
ข้อที่ 7: ประเทศแทนซาเนีย (Tanzania)

คำถาม: เมืองหลวงอย่างเป็นทางการของแทนซาเนียที่ย้ายมาจากเมืองดาร์เอสซาลามคือเมืองอะไร?

คำใบ้: เป็นเมืองหลวงของประเทศที่เป็นที่ตั้งของภูเขาคิลิมันจาโร

คำตอบ: โดโดมา (Dodoma)
//
ข้อที่ 8: ประเทศกานา (Ghana)

คำถาม: เมืองท่าสำคัญที่เป็นเมืองหลวงของประเทศกานาและมีจัตุรัสแบล็กสตาร์คือเมืองอะไร?

คำใบ้: ตั้งอยู่ริมอ่าวกินี

คำตอบ: อักกรา (Accra)
//
ข้อที่ 9: ประเทศแอลจีเรีย (Algeria)

คำถาม: เมืองที่มีฉายาว่า "แอลเจียร์ผู้เลอโฉม" (Algiers the White) เพราะอาคารเป็นสีขาวคือเมืองอะไร?

คำใบ้: ตั้งอยู่ริมทะเลเมดิเตอร์เรเนียน

คำตอบ: แอลเจียร์ (Algiers)
//
ข้อที่ 10: ประเทศเซเนกัล (Senegal)

คำถาม: เมืองหลวงที่ตั้งอยู่ทางตะวันตกสุดของทวีปแอฟริกาคือเมืองอะไร?

คำใบ้: เป็นจุดสิ้นสุดของการแข่งขันแรลลี่ชื่อดังในอดีต

คำตอบ: ดาร์กา (Dakar)
//
